## Expected Conventions

Follow the same patterns used in the Calculus and Linear Algebra courses:
- Week folders: `w1/`, `w2/`, etc.
- Labs in a `Labs/` subfolder, graded work in a `Graded Lab/` subfolder
- Assignment notebooks named `C3_W{n}_Assignment.ipynb` with a `_clean` backup copy
- Unit tests in `w{n}_unittest.py` alongside the assignment
